# BorneAppetitFRONT

Application front-end Angular pour le projet BorneAppetit. Projet généré avec Angular CLI v20.3.5.

## Stack technique

- [Angular](https://angular.io/)
- [TailwindCSS](https://tailwindcss.com/)
- [SpartanUi](https://spartan.ng/)
- [Eslint](https://eslint.org/)

## Prérequis

- Node.js (version LTS recommandée)
- npm (fourni avec Node.js)
- Angular CLI (optionnel) : `npm install -g @angular/cli`

## Installation

Se placer dans le répertoire du projet puis installer les dépendances :

```bash
cd BorneAppetitFRONT
npm install
```

## Démarrer le serveur de développement

Lancer l'application en mode développement :

```bash
ng serve
```

Ouvrir ensuite : http://localhost:4200/ L'application se recharge automatiquement lors des modifications.

## Build (production)

Générer une version optimisée :

```bash
ng build --configuration production
```

Les artefacts sont créés dans dist/.

## Lint

```bash
npm run lint
```
